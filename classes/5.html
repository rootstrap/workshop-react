¿Cómo crear un proyecto en React?

  • Para crear fácilmente un proyecto en React podemos usar una herramienta llamada Create React App (CRA)
  • Para usar esta herramienta necesitamos tener instalado uno de los siguientes manejadores de paquetes: npm o yarn.
  • Una vez que creamos nuestro proyecto con alguno de los comandos provistos en la documentación de CRA, tendremos la siguiente estructura

Implementación del contador en React

  • Vamos a hacer una implementación de un componente Counter:
    • Usando el state (interno al componente)
    • Usando las props (externo al componente)
  • También vamos a estar viendo el uso de eventos en React
